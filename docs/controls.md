# Controls & Gameplay

## Default Controls (Keyboard)
| Action | Key |
|--------|-----|
| Move | WASD |
| Jump | Space |
| Grab / Throw | Left Mouse Button |
| Drop Object | Right Mouse Button |
| Pause / Menu | Esc |

---

## Default Controls (Gamepad)
| Action | Button |
|--------|---------|
| Move | Left Stick |
| Jump | A / Cross |
| Grab / Throw | Right Trigger |
| Drop Object | Left Trigger |
| Pause / Menu | Start / Options |

---

## Gameplay Tips
- Objects gain more force if you spin or sprint before throwing.  
- Environmental hazards reset every round — plan accordingly.  
- Team coordination makes chaos *slightly* more manageable.  
- Keep an eye on weight and object velocity for max yeet power!

---

## Customization
Control bindings can be adjusted in the **Settings** menu (coming soon).  
Support for multiple controllers is automatic via Godot’s InputMap system.
